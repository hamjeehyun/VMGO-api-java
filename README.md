# VMGO-api-java

## 서비스 기능
브이앱의 서비스를 좀 더 즐겁게 즐길 수 있게하는 서비스
- 회원 가입 / 로그인
- 동영상 챌린지 목록 제공
- 동영상 반응
- 사용자가 현재 영상을 얼마나 볼 수 있는지 상태 저장
- 본 동영상 수에 따른 레벨업

## 기술 스택
- JAVA(Open JDK 1.8)
- Spring Boot
- JPA
- Firebase Auth, Spring Security

## 추가 기능
- application.yml profile 분리 하여 관리
- 에러 메세지 slack으로 전송
- GitHub의 GitAction과 AWS Deploy를 이용하여 CI/CD
